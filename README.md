## ✨ 功能特性

### 核心功能

- 📥 **消息下载** - 下载聊天中的图片、视频、文件（免费）
- 📺 **频道下载** - 浏览并下载 Telegram 频道中的媒体文件（高级）
- ⚡ **批量下载** - 支持筛选、按日期范围批量下载（高级）
- 🔓 **突破保存限制** - 支持下载设置了"禁止保存媒体"的频道内容
- 📱 **下载受保护内容** - 支持下载显示灰色图标的受保护媒体文件
- 💾 **本地保存** - 所有文件保存在本地，不上传任何内容，隐私安全

## 📖 使用指南

详细的使用教程请访问：[使用指南](https://td.gewu.dev/guide)

### 快速上手

1. **配置 Telegram API**
   - 访问 [my.telegram.org](https://my.telegram.org/) 获取 API ID 和 API Hash
   - 在软件中填入 API 凭证和手机号

2. **下载消息内容**
   - 在 Telegram 中复制消息链接
   - 粘贴到软件中即可下载

3. **频道批量下载**（高级功能）
   - 选择频道下载功能
   - 设置筛选条件和日期范围
   - 开始批量下载
